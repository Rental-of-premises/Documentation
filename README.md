# Realty Platform

Аналог Авито/Циан для аренды недвижимости.

## Функционал
- Регистрация / вход (пароли шифруются)
- Карточки квартир (фото, параметры)
- Бронирование (статусы: предварительное, ошибочное, установленное, архивное)
- Мессенджер покупателя и продавца - пока в обсуждении, реализация не гарантирована
- Email-уведомления (SMTP)
