# API Структура проекта

## 1. ВХОД

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/auth/sign-up` | Регистрация |
| POST | `/auth/sign-in` | Авторизация |
| POST | `/auth/logout` | Выйти из аккаунта |
| DELETE | `/auth/delete` | Удалить аккаунт |

---

## 2. АККАУНТ

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/account/my-orders` | Мои заказы |
| GET | `/account/my-spaces` | Мои помещения под сдачу |
| GET | `/account/my-chats/buy` | Мои чаты с продавцами |
| GET | `/account/my-chats/sell` | Мои чаты с покупателями |
| PULL | `/account/my-spaces/{space-id}/edit` | Изменить информацию о товаре |
| DELETE | `/account/my-spaces/{space-id}/delete` | Убрать объявление |
| DELETE | `/account/my-orders/{space-id}/delete` | Отменить бронирование |
| GET | `/account/my-orders/{space-id}/status` | Статус брони (есть, нет, отменена, архивирована) |
| GET | `/account/my-orders/achieve` | Архив бронирований со статусами и изменениями |

---

## 3. ВЗАИМОДЕЙСТВИЯ

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/account/new-space` | Добавить помещение под сдачу |
| POST | `/account/new-order` | Добавить бронирование |
| POST | `/chat/message/{order-id}/send` | Написать сообщение в чат при взаимодействии из-за определенного товара |
| POST | `/chat/message/{order-id}/history` | Посмотреть историю сообщений в чате об определенном товаре |
| POST | `/notifications/send` | Отправить уведомление на почту |

---

## 4. КАТАЛОГ

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/catalog` | Посмотреть список всех объявлений |
| GET | `/catalog/{space-id}` | Посмотреть конкретную квартиру |
| GET | `/catalog/{space-id}/reviews` | Посмотреть отзывы о квартире |
